Animal Shelter API Documentation

Test the API: You can test the API endpoints using the Swagger UI.

GET /api/Animals

Retrieves a list of animals from the animal shelter.

Parameters:
  • Animal (string, optional): Filter by animal type ("both", "cat", "dog")
  • Id (integer, optional): Filter by animal ID
  • Breed (string, optional): Filter by animal breed
  • Age (integer, optional): Filter by animal age
  • IsAdopted (string, optional): Filter by adoption status
Response:

Status code 200 - Success

Example:

GET /api/Animals?Animal=cat&Age=2

POST /api/Animals

Creates a new animal in the animal shelter.

Parameters:
  • animal (string): Animal type ("cat" or "dog")
  • name (string): Animal name
  • breed (string): Animal breed
  • age (integer): Animal age
Response:

Status code 200 - Success

Example:

POST /api/Animals

PUT /api/Animals/{id}

Updates an existing animal in the animal shelter.

Parameters:
  • id (integer, path): Animal ID
  • animal (string): Animal type ("cat" or "dog")
  • name (string): Updated animal name
  • breed (string): Updated animal breed
  • age (integer): Updated animal age
  • isAdopted (boolean): Updated adoption status
Response:

Status code 200 - Success

Example:

PUT /api/Animals/10

DELETE /api/Animals/{id}

Deletes an existing animal from the animal shelter.

Parameters:
  • id (integer, path): Animal ID
  • animal (string): Animal type ("cat" or "dog")
Response:

Status code 200 - Success

Example:

DELETE /api/Animals/10?animal=cat
